01 / Services

What I do

I work across the full agentic stack — from workflow design and evaluation through inference, deployment, and the unglamorous parts that decide whether anything actually ships.

01

Agentic workflow design

Multi-step pipelines where each task can be handed off between a human and an agent without rewiring. I design the contracts between steps so the system stays coherent as agent capabilities change underneath it.

02

Cloud & local inference

Equal facility with frontier APIs (Anthropic, OpenAI, Google) and self-hosted open-weight stacks (vLLM, llama.cpp). I choose per workload, not per fashion — with cost, latency, and data residency all on the table.

03

Creative & design pipelines

Asset ingestion, normalization, validation, and generation — battle-tested on Figma plugins and illustration-request workflows that orchestrate concept, mockup, QA, and final delivery across human and agent contributors.

04

Simulation & evaluation

Aerospace and robotics taught me you don't ship what you haven't simulated. I build evaluation harnesses and synthetic environments so agent behavior is measured before production, not after.

02 / About

About me

I'm an engineer whose career has run from aerospace systems on Mars missions through mobile development, robotics, and simulation work, into current research in agentic AI. The throughline: software that has to work the first time, in production, with humans in the loop.

My specialty is agentic systems where every step is structured for seamless handoff between a human and an agent — the only pattern I've seen scale reliably in real environments. I'm equally fluent in cloud LLM APIs and on-prem inference stacks, and I treat simulation-driven evaluation as a non-negotiable, not a luxury.

Engagements run hourly within a budget cap you set, so scope can flex without surprise invoices. I start with a short scoping conversation — sometimes that conversation ends with me telling you agents aren't the right answer for what you're trying to do.
